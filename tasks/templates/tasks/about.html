Your Simple & Powerful Task Manager

PyTasks is a Django-based task management solution that combines essential functionality with a minimalist design. Users can create accounts, organize tasks with categories and priorities, set due dates, and track completion through an intuitive Archives system. The platform features efficient search capabilities and priority filtering to help users maintain focus and productivity through a clean, user-friendly interface.

This full-stack application is built with Django (Python), MySQL, and Bootstrap, featuring secure authentication and responsive design.
